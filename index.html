选题1：Anthropic 15亿美元版权和解：AI训练版权的定价锚落地

【事件链】2024年作者群体起诉Anthropic盗版数百万册书籍训练AI → 2025年Alsup法官初裁认定以书训练属合理使用但盗版获取路径非法 → 2026年7月20日旧金山联邦法官Araceli Martinez-Olguin签署15亿美元集体诉讼和解终局批准，美史上最大版权和解。

【利益格局】原告（作者/出版方）获91%+申领、每部约3000美元；被告Anthropic（Amazon/Alphabet背书）以确定性赔付买断不确定性；全行业LLM厂商（Google/Apple/Nvidia/Perplexity/xAI）同批诉讼被分案处理，各自面对单独版权 liability。

【供需双维】供给端：训练语料版权成本从'灰色免费'变为硬支出，约50万部作品×3000美元=15亿赔付仅覆盖已确权部分，潜在存量语料合规成本陡升；销毁700万+册盗版书库压缩'脏数据'供给。需求端：合规语料（正版电子书/授权语料库）需求与议价权上升，催生版权清算中介市场。

【传导】短期：Alphabet/亚马逊敞口被定价，AI板块情绪正面（不确定性消除）；中期：LLM训练边际成本上移，小模型厂受挤压，利好已建正版语料库者。

【推演】若分案后其他厂商和解金额参照Anthropic倍数，全行业版权赔付或达数十亿级；'合理使用'边界将经多案迭代固化，直接影响开源/闭源模型训练范式。

【暗线】①版权成为AI国力博弈新战场（美监管借司法塑规则）；②数据主权：语料合规化加速各国本土语料池建设；③资本重估：训练成本刚性化削弱'无限缩放'叙事，利好算力高效利用路线。

选题2：霍尔木兹停滞：全球油气供应链的'总开关'再拧紧

【事件链】美伊第九晚打击 → IRGC威胁一桶油气都不让过 → 油轮起火、Brent破90/91 → 约五分之一全球油气经此通道通行近乎停滞。

【利益格局】伊朗以海峡为筹码逼谈判；美方军事施压保通航；油轮/保险/贸易商承担运费与战争险溢价；消费国直面输入型通胀。

【供需双维】供给：全球约20%油气经霍尔木兹，通行停滞直接削减有效供给；需求：短期恐慌补库推升溢价，中长期替代航线（绕好望角）抬升吨海里成本。库存：原油战略储备成唯一缓冲。

【传导】油价↑ → 通胀预期↑ → 美联储加息概率↑ → 股债承压（Nikkei -4.03%已现）；黄金避险买盘。

【推演】若停火10天提议落地，溢价回吐但风险溢价中枢上移；若冲突升级，Brent或挑战110（高盛情景）。

【暗线】①能源安全重构：欧洲/亚洲加速去油气依赖；②航运重构：战争险费率成新成本项；③地缘溢价常态化。

选题3：英伟达反垄断：1467亿罚款风险与全球AI芯片监管收紧

【事件链】法国竞争管理局对英伟达AI芯片业务发起三年反垄断调查 → 拟出具指控文书（据路透社早前报道，最高罚1467亿元） → 若落地成全球首例AI芯片反垄断正式诉讼。

【利益格局】监管方（法/欧/美多经济体）借个案收紧AI芯片反垄断审查；英伟达（CUDA生态）面临'单一厂商垄断'定性风险；竞品（AMD/TSMC/ASML及国产算力）获多元生态窗口。

【供需双维】供给：若处罚落地，英伟达供货/生态合作受审视，短期算力供给不确定性上升；需求：客户（云厂/主权AI）加速多供应商策略，非英伟达架构需求抬升。

【传导】股价波动（反垄断叙事压制估值）→ 全球AI芯片监管预期收紧 → 算力投资节奏重估。
